# IR LED 驱动电路设计文档

## 方案概述

**USB Type-C 5V 输入 → MT3608 升压至 9V → PT4205 降压恒流驱动 → 红外 LED（1050–1550 nm）**

由于 PT4205 最低输入电压为 6 V，不能直接由 USB Type-C 5 V 供电，因此在前级增加 MT3608
升压模块，先将 5 V 升至 9 V，再供给 PT4205 进行恒流控制。

---

## 模块说明

### 模块 A：USB Type-C 输入

| 元件 | 说明 |
|------|------|
| USB1 | TYPE-C 16PIN 2MD(073) 连接器 |
| R_CC1 | 5.1 kΩ，CC1（Pin 4）下拉至 GND |
| R_CC2 | 5.1 kΩ，CC2（Pin 11）下拉至 GND |

> **重要**：CC1/CC2 必须各接 5.1 kΩ 下拉电阻，否则主机（充电器/电脑）不会在 VBUS 上提供电源。

- VBUS → 网络标号 `VIN_5V`
- SHIELD → GND

### 模块 B：MT3608 升压模块（5 V → 9 V）

| 元件 | 说明 |
|------|------|
| U2 | MT3608，SOT-23-6 |
| L2 | 10 μH 升压电感，连接 IN 与 SW |
| D2 | SS14 续流二极管，阳极接 SW，阴极接 9 V 输出 |
| C3 | 10 μF/25 V 输入旁路电容（IN 对 GND） |
| C4 | 22 μF/25 V 输出滤波电容（VOUT 对 GND） |
| R3 | 140 kΩ，反馈分压上电阻（VOUT 到 FB） |
| R4 | 10 kΩ，反馈分压下电阻（FB 到 GND） |

- 输出网络标号：`VBUS_9V`

### 模块 C：PT4205 恒流驱动（9 V 输入）

| 元件 | 说明 |
|------|------|
| U1 | PT4205，SOT-89-5 |
| C1, C2 | 220 μF 输入电解电容（各 1 只，并联） |
| L1 | 47 μH 电感 |
| D1 | SS14 续流二极管 |
| R1, R2 | 560 mΩ 采样电阻（各 1 只，并联 = 280 mΩ） |
| J1 | OUT+/OUT- LED 输出接口 |
| J2 | DIM 调光控制接口（PWM 或模拟） |

## 电路工作原理

### MT3608 升压原理

MT3608 是同步/非同步 Boost DC-DC 控制器，内置 4 A MOSFET。工作过程：

1. 内部 MOSFET 导通时，电流经 L2 从 IN 流向 GND，电感储能。
2. MOSFET 关断时，L2 的感应电动势叠加在输入电压上，经 D2 向输出电容 C4 充电，
   产生高于输入的输出电压。
3. FB 引脚检测 R3/R4 分压（基准 0.6 V），实现输出电压闭环调节。
4. EN 接 VIN，芯片始终使能。

### PT4205 降压恒流原理

PT4205 是恒频电流模式降压 LED 驱动器：

1. 内部 MOSFET 导通时，电流经 L1 流向 LED 串，同时在采样电阻 R1‖R2 两端产生压差。
2. CSN 引脚监测采样电阻电压，当该电压达到 100 mV（内部基准）时关断 MOSFET。
3. MOSFET 关断后，L1 续流电流经 D1 继续向 LED 供电，直至下一周期再次开通。
4. DIM 引脚支持 PWM 调光（10 Hz–10 kHz）或模拟调光（0.3–2 V）。

## 关键参数计算

### MT3608 输出电压

```
Vout = Vref × (1 + R3/R4)
     = 0.6 V × (1 + 140 kΩ / 10 kΩ)
     = 0.6 V × 15
     = 9 V
```

### PT4205 LED 电流

```
I_LED = Vcs / R_sense
      = 0.1 V / (R1 ‖ R2)
      = 0.1 V / (560 mΩ / 2)
      = 0.1 V / 280 mΩ
      ≈ 357 mA
```

## 注意事项

1. **USB CC 下拉电阻**：R_CC1 和 R_CC2 是必须元件。在 USB Type-C 电阻供电协商中，
   CC 引脚的 5.1 kΩ 下拉电阻向主机表明该设备是用电设备（Sink），主机据此在 VBUS
   上提供电源。缺少这两个电阻，绝大多数 USB 充电器和 PC USB 接口不会输出 VBUS 电压。
   5.1 kΩ 下拉对应默认 USB 电流（≥ 500 mA）；若主机支持 1.5 A / 3 A 模式，
   可分别将 CC 下拉改为 22 kΩ（1.5 A Rp）或 10 kΩ（3 A Rp）——但通常 5.1 kΩ
   对驱动本电路已经足够。

2. **布局建议**：
   - MT3608 的 L2、D2、C3、C4 应紧靠芯片，走线尽量短粗。
   - PT4205 的 L1、D1 应靠近芯片 SW 引脚，电流回路面积最小化。
   - 采样电阻 R1/R2 布置在 GND 铜皮上，底层铺铜散热。
   - C1/C2 电解电容放置靠近 PT4205 VIN 引脚。

3. **散热**：PT4205 SOT-89 封装底部裸露焊盘需焊接到 PCB 铜皮上以增强散热。

4. **调光接口 J2**：DIM 引脚若不使用 PWM 调光，可通过 J2 接 3.3 V 或 5 V（经限流
   电阻），或直接接高电平以全亮运行。

5. **LED 极性**：J1 Pin1 为 OUT+（阳极侧），Pin2 为 OUT-（阴极经采样电阻到 GND）。
   红外 LED 串的正极接 OUT+，负极接 OUT-。
